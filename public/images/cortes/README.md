# Imagenes por corte

Sube aqui una imagen por cada corte usando estos nombres exactos:

- `bife-chorizo.jpg`
- `ojo-bife.jpg`
- `lomo.jpg`
- `entrana.jpg`
- `tira-asado.jpg`
- `tomahawk.jpg`
- `t-bone.jpg`

Notas:

- Puedes usar `.jpg`, `.jpeg` o `.png`, pero si cambias la extension tambien debes ajustar la ruta en `app/components/CowDiagram.tsx`.
- Mientras faltan imagenes, el modal usa fallback automatico a `/images/carne1.png`.
